# KoKoFriendList
---
A simple test APP that simulates the KoKo APP friend list page.

## Environment
---
Xcode 16.1 +

## Features 
---
透過api控制模擬以下好友列表情境
- 1.無好友
- 2.只有好友列表
- 3.含好友列表和邀請列表
